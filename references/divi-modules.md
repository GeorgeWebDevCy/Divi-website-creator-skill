# Divi Module Reference

Comprehensive guide to Divi modules and their use cases.

## Content Modules

### Text Module
- **Purpose**: Basic text content
- **Settings**: Text, Background, Text styling (font, size, color, line height)
- **Advanced**: Custom CSS, CSS ID/Class, Visibility

### Image Module
- **Purpose**: Display images
- **Settings**: Image source, Alt text, Alignment, Animation
- **Advanced**: Overlay, Box shadow, Border, Filters

### Blurb Module
- **Purpose**: Icon + title + text combination
- **Settings**: Icon/Image, Title, Body text, Layout
- **Use for**: Features, services, benefits

### Button Module
- **Purpose**: Call-to-action links
- **Settings**: Text, URL, Alignment, Custom styles
- **Styles**: Fill, Outline, Custom colors, Hover effects

## Structure Modules

### Divider Module
- **Purpose**: Visual separation
- **Styles**: Solid, Dashed, Dotted, Gradient
- **Sizing**: Height, Width

### Spacer Module
- **Purpose**: Add vertical space
- **Use for**: Controlling whitespace between elements

## Media Modules

### Video Module
- **Purpose**: Embed video content
- **Sources**: Self-hosted, YouTube, Vimeo
- **Settings**: Autoplay, Loop, Mute

### Audio Module
- **Purpose**: Embed audio files
- **Settings**: Cover image, Title, Artist

## Marketing Modules

### Call To Action (CTA)
- **Purpose**: Prominent call-to-action
- **Elements**: Title, Body, Button, Background
- **Use for**: Landing pages, promotions

### Email Optin
- **Purpose**: Email list building
- **Integrations**: MailChimp, AWeber, etc.
- **Fields**: Email, Name (optional)

### Contact Form
- **Purpose**: User inquiries
- **Fields**: Customizable
- **Features**: CAPTCHA, Redirect

## Social Modules

### Social Media Follow
- **Purpose**: Link to social profiles
- **Networks**: Facebook, Twitter, Instagram, LinkedIn, etc.
- **Styles**: Icon, Icon + Text, Follow button

### Social Media Share
- **Purpose**: Share current page
- **Networks**: Configurable
- **Placement**: Inline or floating

## Advanced Modules

### Tabs
- **Purpose**: Organized content sections
- **Use for**: Product info, FAQs, multi-step content

### Accordion
- **Purpose**: Collapsible content sections
- **Use for**: FAQs, long content lists

### Toggle
- **Purpose**: Single collapsible section
- **Use for**: Additional info, spoilers

### Map
- **Purpose**: Embed Google Maps
- **Settings**: Location, Zoom, Map type

### Countdown Timer
- **Purpose**: Create urgency
- **Use for**: Launches, sales, events

### Portfolio
- **Purpose**: Showcase projects
- **Layouts**: Grid, Masonry, Fullwidth
- **Filters**: Category-based filtering

### Testimonial
- **Purpose**: Customer quotes
- **Elements**: Quote, Author, Company, Image

### Team Member
- **Purpose**: Staff profiles
- **Elements**: Photo, Name, Position, Bio, Social links

### Pricing Table
- **Purpose**: Product/service pricing
- **Features**: Highlighted plan, Feature lists, CTAs

### Blog
- **Purpose**: Display posts
- **Layouts**: Fullwidth, Grid, Masonry
- **Filters**: Categories, tags, recent, featured

## WooCommerce Modules

### Shop Module
- **Purpose**: Display products
- **Layouts**: Grid, List, Slider

### Product
- **Purpose**: Single product display
- **Use for**: Featured products

### Cart
- **Purpose**: Shopping cart

### Checkout
- **Purpose**: Checkout form

## Module Configuration Tips

1. **Design Tab**: Visual styling (colors, fonts, spacing, borders)
2. **Content Tab**: Main content and settings
3. **Advanced Tab**: CSS, animations, visibility conditions

### Spacing
- **Margin**: Space outside the module
- **Padding**: Space inside the module

### Responsive
- **Desktop/Tablet/Mobile**: Customize per breakpoint
- **Hide on**: Conditionally hide modules

### Animations
- **Style**: Fade, Slide, Zoom, etc.
- **Trigger**: On scroll, hover, etc.
- **Duration/Timing**: Control animation feel
